# @tansuk/rott-ui

This is property based full built-in UI Kit for React-Native

## Installation

```sh
npm install @tansuk/rott-ui
```

## Usage

```js
import {multiply} from '@tansuk/rott-ui'

// ...

const result = await multiply(3, 7)
```
